Данная программа разработана для более удобного заучивания японских слов. В целом работа с ней строится следующим образом - сначала пользователь выбирает тип теста (опроса), а затем собственно идет сам процесс тестирования (опроса).
Для корректной работы программы рядом с файлом приложения, т.е. в одной с ним папке, должны лежать следующие приложенные файлы - N5_goi.dat, dict_kanji.dat, ReadMe.txt.

Тип теста выбирается только один. Впоследствии его можно сменить. Список доступных режимов тестов можно увидеть в левой части окна приложения. К примеру, "слова на хирагане -> русский" означает, что пользователю нужно писать русский перевод японских слов, написанных хираганой.
После этого необходимо указать, с номера какого слова и в каком количестве будет вестись опрос. Эти числа не может быть равно нулю, разумеется. Например, "Начиная с: 250, Количество слов: 300" составит список с 250 по 550 слова. В процессе теста после правильных ответов число слов будет уменьшаться, и когда оно достигнет нуля, тест будет считаться завершенным. Номера слов смотрите в нужных таблицах.
Также можно воспользоваться опцией "Слова разбросом". Так в тест попадут случайные слова из таблицы в заданном количестве.
Далее нужно нажать кнопку "начать". Ответы вводятся в поле рядом с кнопкой "ОК". После  ввода ответа и его подтверждения будет выведено сообщение, правильный ответ или нет. Если ответ неправильный, под счетчиком оставшихся слов, рядом с надписью "Ответ" будет указан верный ответ.
Если Вы не хотите видеть правильные ответы (или хотите ускорить реакцию приложения), Вы можете отключить эту ф-цию во вкладке "вид".
В этой же вкладке можно изменять отображение бонусной картинки (была добавлена для разнообразия).
